# Tech Stack: bqopd

## Frontend
- **Language:** Dart
- **Framework:** Flutter
- **State Management:** Provider
- **Navigation:** GoRouter

## Backend
- **Platform:** Firebase
- **Authentication:** Firebase Auth
- **Database:** Cloud Firestore
- **Storage:** Cloud Storage for Firebase
- **Functions:** Cloud Functions for Firebase (Python)

## Tools & Libraries
- **Image Processing:** image_picker
- **Permissions:** permission_handler
- **Other:** intl, nanoid2, firebase_analytics, url_launcher
